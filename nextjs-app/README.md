# freitagskind Website Redesign

A stunning redesign of the freitagskind website using **taste-skill** design principles.

## What's New

### Premium Design Features
- **Asymmetric layouts** - Split-screen hero instead of centered
- **Geist font** - Premium typography (Google Fonts)
- **Smooth animations** - Framer Motion with spring physics
- **Single accent color** - #D97706 amber (no AI purple!)
- **Spacious design** - Generous whitespace, breathing room
- **Floating elements** - Animated decorative cards
- **Glassmorphism** - Backdrop blur navigation
- **Smooth hover interactions** - Scale, translate effects

### Content
- **Hero:** "Apps that get out of the way."
- **Apps showcase:** Tab (iOS), Coming Soon
- **Principles:** Calm over clever, No tricks, Made with care
- **Imprint page:** Legal information

### Technology
- **Next.js 16** - React framework
- **React 19** - UI library
- **TypeScript** - Type safety
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Custom CSS** - Reliability over Tailwind (Turbopack issue)

## Getting Started

```bash
cd nextjs-app
npm install
npm run dev
```
